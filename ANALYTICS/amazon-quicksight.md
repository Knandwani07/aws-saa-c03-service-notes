# Amazon QuickSight

Amazon QuickSight is a fully managed business intelligence (BI) service that enables you to easily create and publish interactive dashboards, visualizations, and analyses from various data sources.
It is designed to be scalable, serverless, and cost-effective, with pay-per-session pricing for readers.

---

## What It Is

• Managed, serverless BI service with no infrastructure to manage  
• Interactive dashboards with rich visualizations and drill-downs  
• Supports ad hoc analysis and sharing of insights  
• Pay-per-session pricing model for readers  
• SPICE (Super-fast, Parallel, In-memory Calculation Engine) for fast performance  
• Supports scheduled and email reports  
• ML Insights for anomaly detection and forecasting  
• Integration with AWS and on-premises data sources  

---

## Key Features

• Fully managed, serverless architecture  
• Fast in-memory analytics using SPICE  
• Supports embedding dashboards into applications  
• Scheduled reporting and email distribution  
• ML Insights for anomaly detection and forecasting  
• Row-level security controls  
• Enterprise authentication with IAM, SSO, and Active Directory  

---

## Data Sources

• AWS services: S3, Athena, Redshift, RDS, Aurora, EMR, Glue Data Catalog  
• On-premises databases via QuickSight data gateway  
• Third-party databases: MySQL, PostgreSQL, SQL Server, Oracle, Teradata  
• Files: CSV, Excel, JSON  
• SaaS connectors: Salesforce, ServiceNow, Twitter, and others  

---

## SPICE Engine

• In-memory storage and calculation engine  
• Enables fast, interactive querying on large datasets  
• Automatically scales to support thousands of users  
• Data refresh can be scheduled or on-demand  
• More cost-effective than direct querying of databases  

---

## Dashboard and Visualization

• Rich visuals: bar, line, pie, maps, tables, KPIs  
• Drill-down and drill-through functionality  
• Parameterized dashboards for personalization  
• Filters and controls for interactive exploration  
• Dashboards can be embedded in apps and portals  

---

## ML Insights

• Built-in ML capabilities with no ML expertise required  
• Anomaly detection to identify unexpected changes  
• Forecasting for predictive trends  
• Narrative Insights for automatic, text-based summaries  

---

## User Management and Access Control

• Authentication via AWS IAM  
• Role-based access control within QuickSight  
• Integration with AWS SSO and Active Directory  
• Row-level security for restricting dataset access  

---

## Deployment Options

• **QuickSight Standard Edition** – basic dashboards for individuals/teams  
• **QuickSight Enterprise Edition** – advanced security, ML insights, enterprise integration  
• Pay-per-session model for readers  
• SPICE capacity purchased separately  

---

## Security

• Encryption at rest using AWS KMS  
• Encryption in transit using SSL/TLS  
• VPC connectivity for secure private data source access  
• Fully compliant with AWS security and compliance standards  

---

## Integration with AWS Services

• Redshift, Athena, S3, RDS, Aurora  
• AWS Glue Data Catalog for metadata management  
• AWS IoT Analytics  
• Embedding dashboards via AWS SDKs  

---

## Monitoring and Management

• CloudWatch for usage metrics and alarms  
• Usage tracking for pay-per-session billing  
• CloudTrail for audit logging  
• Centralized management of users and permissions  

---

## Pricing

• Authors: fixed monthly fee per author  
• Readers: pay-per-session or monthly pricing  
• SPICE storage priced per GB per month  
• No infrastructure costs

## Quick Summary

Amazon QuickSight is a serverless BI service that delivers fast, interactive dashboards using the SPICE engine.
It integrates with various AWS and external data sources, supports cost-efficient pay-per-session pricing, and includes ML-powered insights for anomaly detection and forecasting.
