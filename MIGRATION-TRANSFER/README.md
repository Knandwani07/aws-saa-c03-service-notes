# üìÅ Services Included in This Category  
(Ordered **exactly** according to the File Index)

| Service | What It Does |
|---------|--------------|
| **AWS Application Discovery Service** | Scans on-prem servers to collect usage, performance, and dependency data for migration assessment and planning. |
| **AWS Application Migration Service (MGN)** | Lift-and-shift (rehost) service that continuously replicates on-prem servers to AWS with minimal downtime. |
| **AWS DataSync** | High-speed, automated data transfer between on-prem storage and AWS services (S3, EFS, FSx). Includes encryption and performance acceleration. |
| **AWS Database Migration Service (DMS)** | Migrate databases to AWS with near-zero downtime. Supports both homogeneous and heterogeneous migrations. |
| **AWS Snowmobile** | Exabyte-scale data migration using a ruggedized semi-truck for offline transfer to AWS. |
| **AWS Snowball Edge** | Rugged, portable devices for large-scale or offline data transfers and edge compute workloads. Part of the AWS Snow Family. |
| **AWS Transfer Family** | Managed SFTP, FTPS, and FTP service to transfer files directly into and out of Amazon S3 or EFS. |
