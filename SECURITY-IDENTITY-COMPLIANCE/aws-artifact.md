# AWS Artifact

---

## What It Is
AWS Artifact is a **self-service compliance and audit portal** that provides on-demand access to AWS’s security, compliance, and regulatory documents. It includes downloadable audit reports and legally binding agreements required for running regulated workloads on AWS.

---

## Key Features

### **Artifact Reports**
Provides downloadable compliance documents including:  
• SOC 1, SOC 2, SOC 3  
• ISO certifications  
• PCI DSS  
• Other third-party audit reports  
These verify AWS's adherence to industry compliance standards.

### **Artifact Agreements**
Allows customers to review and accept AWS legal agreements such as:  
• **HIPAA Business Associate Addendum (BAA)**  
• **GDPR Data Processing Addendum (DPA)**  
These agreements enable customers to store and process regulated data in AWS.

---

## Key Benefits
• Full transparency into AWS’s compliance posture  
• Supports audits, risk assessments, and regulatory reporting  
• Self-service—no need to contact AWS Support  
• Helps organizations validate AWS compliance before migrating sensitive workloads  

---

## Security & Access
• IAM policies specify who can view or download documents  
• Actions are logged in **AWS CloudTrail** for audit tracking  
• Agreements can only be accepted by authorized users in the account  

---

## Use Cases
• Auditors reviewing AWS compliance documentation  
• Organizations needing formal proof of compliance (e.g., SOC reports)  
• Companies requiring acceptance of HIPAA BAA or GDPR DPA before using AWS for regulated data  
• Supporting internal compliance, governance, and security teams  

---

## Pricing
• AWS Artifact is **free**.

## Exam Tips
• AWS Artifact = portal for **compliance reports + agreements**  
• Use it for SOC, ISO, PCI audit reports  
• Accept HIPAA BAA or GDPR DPA using Artifact Agreements  
• IAM controls access; CloudTrail logs activity  
• Fully self-service; no support ticket needed  

---

## Quick Summary
AWS Artifact is a free compliance portal providing access to AWS audit reports (SOC, ISO, PCI) and legal agreements (HIPAA BAA, GDPR DPA). It helps organizations meet security, regulatory, and audit requirements with self-service access and full IAM + CloudTrail integration.
