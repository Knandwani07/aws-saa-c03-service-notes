# üìÅ Services Included in This Category  
(Ordered **exactly** according to the File Index)

| Service | What It Does |
|---------|--------------|
| **Amazon Cognito** | Manages user authentication and access for web/mobile apps with sign-up, sign-in, MFA, and identity federation. |
| **Amazon Detective** | Investigates security findings using graphs of users, events, and resources to simplify root-cause analysis. |
| **Amazon GuardDuty** | Intelligent threat detection service that uses ML and AWS logs to identify anomalies and suspicious activities. |
| **Amazon Inspector** | Automated vulnerability scanning for EC2, Lambda, and container images. Identifies CVEs and risks. |
| **Amazon Macie** | ML-powered sensitive data discovery (PII) for Amazon S3 with automated classification and security alerts. |
| **AWS Artifact** | Provides on-demand access to AWS compliance reports (SOC, PCI, ISO) and agreements. |
| **AWS Audit Manager** | Continuously collects evidence to support audits and compliance requirements. |
| **AWS Certificate Manager (ACM)** | Issues, manages, and renews SSL/TLS certificates. Integrates with ELB, CloudFront, API Gateway. |
| **AWS CloudHSM** | Dedicated hardware security modules for organizations requiring full control of encryption keys. |
| **AWS Directory Service** | Managed Active Directory, AD Connector, and Simple AD for integrating identity with AWS workloads. |
| **AWS Firewall Manager** | Centralized security policy management for WAF, Shield Advanced, and Network Firewall. |
| **AWS IAM** | Identity and access control for AWS. Define users, roles, groups, policies, permissions. Core security service. |
| **AWS KMS** | Managed key management service for encryption keys, integrated across AWS for data protection. |
| **AWS Network Firewall** | Managed network-level firewall for VPCs with deep packet inspection and traffic filtering. |
| **AWS Resource Access Manager (RAM)** | Securely share resources (subnets, Transit Gateway, License Manager configs) across AWS accounts. |
| **AWS Secrets Manager** | Managed secrets storage with rotation for API keys, DB credentials, and tokens. |
| **AWS Security Hub** | Centralized security posture dashboard aggregating findings from GuardDuty, Inspector, Macie, IAM Access Analyzer, etc. |
| **AWS Shield** | DDoS protection service. Standard included; Shield Advanced provides enhanced protection and response. |
| **AWS WAF** | Web application firewall protecting against SQL injection, XSS, bots, and common exploits. |
